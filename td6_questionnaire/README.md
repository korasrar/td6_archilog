# sae_devmobile

## Installation

```bash
cd td6_questionnaire/
npm install
python3 -m venv venv
source venv/bin/activate
pip install -r requirements
```

### Lancement de l'api
```bash
flask run
```

### Lancement du client

```bash
npm run dev
```
